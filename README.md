# Pizzaria

<br/>

**Descrição:**
Este projeto é um site de pizzaria com cardápio digital e carrinho de compras, desenvolvido para proporcionar uma experiência moderna e prática. Quando o usuário clica na pizza desejada, aparece uma janela onde ele pode escolher o tamanho (pequena, média ou grande) e a quantidade de pizzas que deseja. Depois de escolher, basta adicionar ao carrinho. Conforme o usuário vai adicionando ou removendo pizzas, o valor total do carrinho é atualizado automaticamente, mostrando o subtotal, desconto e total final em tempo real. O objetivo é facilitar o pedido, tornando o processo intuitivo e rápido, tanto no computador quanto no celular.

<br/>

**Acesse o site em:** https://projeto-pizzaria-da-nonna-monanleal.netlify.app/

<br/>
<br/>

### **Como o JavaScript atua na página:**

1. **Leitura dos Dados**: O array pizzaJson serve como fonte de dados para o cardápio.
2. **Renderização Dinâmica**: O JavaScript percorre esse array e, para cada pizza, gera dinamicamente elementos HTML que exibem as informações na tela.
3. **Exibição dos Tamanhos e Preços**: Para cada pizza, são mostrados os tamanhos disponíveis e seus respectivos preços, permitindo ao usuário escolher conforme sua preferência.
4. **Imagens e Descrições**: Cada pizza exibe uma imagem ilustrativa e uma descrição dos ingredientes, tornando a experiência mais visual e informativa.
5. **Facilidade de Expansão**: A estrutura do array permite adicionar, remover ou editar sabores facilmente, bastando alterar o pizzaJson.
